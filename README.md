# 🔐 Privacy Center – Android Security Toolkit

Privacy Center es una aplicación Android desarrollada en Kotlin que analiza distintos aspectos de la seguridad digital del usuario:

- 🔍 Filtraciones de datos (data leaks)
- 📡 Seguridad de red (WiFi analysis)
- 🕵️ Detección de trackers web

El objetivo del proyecto es demostrar habilidades en desarrollo Android moderno (Jetpack Compose), consumo de APIs y diseño de interfaces con feedback visual en tiempo real.

## 🚀 Features

### 🔎 Data Leak Scanner
- Consulta filtraciones mediante API pública
- Validación de email
- Visualización de resultados en tiempo real

### 📡 Network Security Analyzer
- Detección de tipo de conexión (WiFi / Datos móviles)
- Evaluación de riesgo con scoring (0–100)
- Detección de VPN
- Geolocalización IP (ciudad, país, ISP)
- UI interactiva con animaciones

### 🕵️ Tracker Detection
- Análisis de posibles trackers en sitios web
- Score de riesgo (Low / Medium / High)
- Interfaz estilo “cyber security”
